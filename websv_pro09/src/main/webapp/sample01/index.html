배열

배열 기본 접근 방법

indexOf 함수 - 인덱스번호 반환

push 함수 - 배열뒤에 추가로 붙임(''사용하면 문자열로 들어감)

unshift 함수 - 배열앞에 추가로 붙임

pop 함수 - 맨 뒤 요소를 제거(반환도 함)

shift 함수 - 맨 앞 요소를 제거(반환도 함)

reverse 함수 - 배열 역순 정렬

sort 함수 - 배열 정렬(오름차순, 내림차순)

slice 함수 - 시작index와 끝 index에 포함된 요소만 잘라옴

concat 함수 - 배열결합

toString 함수 - 문자열로 반환

join 함수 - ""안에 있는값을 기준으로 결합 / 문자열로 반환

문제 1

다음의 input 요소에 있는 값을 배열로 만들어 exam1 에 출력
출력 형식은 ['a', 'b', 'c', 'd', 'e'] 와 같이 대괄호 안에 값이 출력되도록 한다.

문제 2

다음의 input 요소에 있는 값을 문제 1에서 만든 배열에 추가후 출력
출력 형식은 문제 1과 동일하다. 추가로 input 요소에 있는 값의 총 합을 구하여 배열에 추가도 한다.
