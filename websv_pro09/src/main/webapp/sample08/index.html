Jquery

기본 사용 방법

$(CSS선택자).메서드([매개변수 [, 매개변수, [...]]]);

  • $() : Jquery를 사용하기 위해 기본적으로 사용하는 문법
  • .메서드 : Jquery에서 제공하는 다양한 메서드를 작성
  • 매개변수 : 메서드에서 요구하는 전달값, 메서드에 따라 다양한 매개변수 사용됨

사용 형식에서 [] 대괄호가 있는 경우 생략할 수 있다는 의미를 가진다.

$(CSS선택자)

  • $(".jq-selector")
  • $("#ul")
  • $("#ul1 > li")

순회(탐색) - Traversing

filtering 관련 메서드

선택자로 지정한 객체를 기준으로 객체 그룹에서 위치를 가지고 선택하는 메서드

  • $(CSS선택자).first();
  • $(CSS선택자).eq(index);
  • $(CSS선택자).filter(CSS선택자);
  • $(CSS선택자).not(CSS선택자);
  • $(CSS선택자).last();

ancestors 관련 메서드

선택자로 지정한 객체를 기준으로 상위 요소들을 선택할 수 있는 메서드

  • $(CSS선택자).parent()
  • $(CSS선택자).parents()
  • $(CSS선택자).parentsUntil()

descendants 관련 메서드

선택자로 지정한 객체를 기준으로 하위 요소들을 선택할 수 있는 메서드

  • $(CSS선택자).children([CSS선택자])
  • $(CSS선택자).find(CSS선택자)

sideways 관련 메서드

선택자로 지정한 객체를 기준으로 형제/자매 요소들을 선택할 수 있는 메서드

  • $(CSS선택자).siblings([CSS선택자])
  • $(CSS선택자).next()
  • $(CSS선택자).nextAll()
  • $(CSS선택자).nextUntil(CSS선택자)
  • $(CSS선택자).prev()
  • $(CSS선택자).prevAll()
  • $(CSS선택자).prevUntil(CSS선택자)

is 관련 메서드

선택자로 지정된 범위에 특정 요소가 존재하는지 찾을 때 사용 메서드

  • $(CSS선택자).is(CSS선택자)
